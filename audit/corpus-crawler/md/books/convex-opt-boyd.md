**A MOOC on convex optimization, [CVX101](https://class.stanford.edu/courses/Engineering/CVX101/Winter2014/about), was run from 1/21/14 to 3/14/14**. If you register for it, you can access all the course materials.

More material can be found at the web sites for [EE364A](https://web.stanford.edu/class/ee364a) (Stanford) or [EE236B](http://www.seas.ucla.edu/~vandenbe/ee236b/ee236b.html) (UCLA), and our own web pages. Source code for almost all examples and figures in part 2 of the book is available in [CVX](http://cvxr.com/cvx/) (in the [examples directory](http://cvxr.com/cvx/examples/)), in [CVXOPT](http://cvxopt.org/) (in the book examples directory), and in [CVXPY](http://cvxpy.org/). Source code for examples in Chapters 9, 10, and 11 can be found [here](cvxbook_examples/). Instructors can obtain complete solutions to exercises by email request to us; please give us the URL of the course you are teaching.

If you find an error not listed in our [errata list](cvxbook_errata.html), please do let us know about it.

[Stephen Boyd](http://www.stanford.edu/~boyd/) & [Lieven Vandenberghe](http://www.ee.ucla.edu/~vandenbe/)
